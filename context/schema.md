# Schéma JSON d'extraction

## Structure attendue pour chaque rawi

```json
{
  "رقم_الترجمة": "الرقم العربي في بداية الترجمة",
  "الاسم_الكامل": "الاسم كما هو في النص",
  "الكنية": "أبو... أو أم... إن وُجدت",
  "النسبة": "النسبة الجغرافية أو القبلية إن وُجدت",
  "الكتب": ["اختصارات الكتب المذكورة بعد الرقم مباشرة"],
  "الشيوخ": ["أسماء من بعد رَوَى عَن فقط"],
  "التلاميذ": ["أسماء من بعد رَوَى عَنه فقط"],
  "أقوال_العلماء": [
    {
      "العالم": "اسم العالم كما في النص",
      "القول": "نص قوله بالضبط كما في النص"
    }
  ],
  "وفاته": "نص ذكر الوفاة كما في النص"
}
```

## قواعد صارمة
1. استخرج فقط ما هو مكتوب في النص — لا تضف شيئاً
2. إذا لم يُذكر حقل، اتركه فارغاً `""` أو `[]`
3. انقل الأسماء والأقوال بالضبط كما هي في النص
4. الجرح والتعديل يجب أن يُنسب لعالم مذكور في النص

## مثال على ترجمة صحيحة

**النص:**
```
١٣٨ - بخ م س ق: أبان بن صعصعة الأنصاري البصري.
روى عن: أبي الوازع جابر بن عمرو الراسبي، وشهر بن حوشب.
روى عنه: خالد بن الحارث، ويحيى بن سعيد القطان.
قال يحيى بن معين: ثقة.
قال أحمد بن حنبل: صالح.
مات سنة ثلاث وخمسين ومئة.
```

**الاستخراج الصحيح:**
```json
{
  "رقم_الترجمة": "١٣٨",
  "الاسم_الكامل": "أبان بن صعصعة الأنصاري البصري",
  "الكنية": "",
  "النسبة": "الأنصاري، البصري",
  "الكتب": ["بخ", "م", "س", "ق"],
  "الشيوخ": ["أبي الوازع جابر بن عمرو الراسبي", "شهر بن حوشب"],
  "التلاميذ": ["خالد بن الحارث", "يحيى بن سعيد القطان"],
  "أقوال_العلماء": [
    {"العالم": "يحيى بن معين", "القول": "ثقة"},
    {"العالم": "أحمد بن حنبل", "القول": "صالح"}
  ],
  "وفاته": "مات سنة ثلاث وخمسين ومئة"
}
```
