# تهذيب الكمال في أسماء الرجال

## المؤلف
يوسف بن عبد الرحمن المزي (توفي 742 هـ)

## وصف الكتاب
كتاب تهذيب الكمال هو أشمل كتب علم الرجال وأدقها. يترجم لرواة الكتب الستة (البخاري، مسلم، أبو داود، الترمذي، النسائي، ابن ماجه) ويذكر لكل راوٍ:
- اسمه الكامل وكنيته ونسبته
- الكتب الستة التي روى لها
- شيوخه الذين روى عنهم
- تلاميذه الذين رووا عنه
- أقوال العلماء فيه (الجرح والتعديل) مع نص القول بالضبط
- تاريخ وفاته

## شكل الترجمة في النص
كل ترجمة تبدأ بهذا الشكل:
```
[رقم] - [اختصارات الكتب]: [الاسم الكامل]
```

**أمثلة:**
```
١٣٨ - بخ م س ق: أبان بن صعصعة الأنصاري البصري
١ - د فق: أحمد بن إبراهيم بن خالد الموصلي
٢٠ - م د س: أحمد بن جناب المصيصي
```

## اختصارات الكتب الستة
| الاختصار | الكتاب |
|---|---|
| خ أو بخ | صحيح البخاري |
| م | صحيح مسلم |
| د أو أد | سنن أبي داود |
| ت | جامع الترمذي |
| س أو ن | سنن النسائي |
| ق أو جه | سنن ابن ماجه |
| ٤ | الأربعة (أبو داود + ترمذي + نسائي + ابن ماجه) |
| خت | البخاري تعليقاً |
| مد | أبو داود في المراسيل |

## بنية النص داخل الترجمة
1. **رَوَى عَن:** → قائمة الشيوخ
2. **رَوَى عَنه:** → قائمة التلاميذ
3. **قال [اسم العالم]:** → أقوال الجرح والتعديل
4. **مات / توفي / وفاته** → تاريخ الوفاة
